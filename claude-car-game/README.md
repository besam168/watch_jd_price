# Claude Car Game

A tiny browser car-dodging game made with plain HTML/CSS/JavaScript.

## Run

1. Open `index.html` directly in your browser.
2. Play immediately (no install/build needed).

## Controls

- `←` / `→`: move car left/right
- `Space`: restart after game over

## Rules

- Avoid incoming obstacle cars.
- Score increases over time.
- Collision ends the game.
