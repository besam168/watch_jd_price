# 林泽琪成长纪念册 · Expert Designer 增强版测试配置

基于原始文件：
- `skills/pptx-generator/output/linzeqi-totoro-finished.json`

当前定位：
- 用昨天的真实 PPT 项目，验证 `design-expert-mode` 是否能把内容稿提升为更完整的设计稿。

## 本轮增强目标

### 1. 封面页
- 保留森林系 / 龙猫气质 / 成长感
- 强化标题安全区
- 强化副标题与主视觉的呼吸感
- 封面文案块减少“信息堆砌感”

### 2. 目录页
- 从单列 bullet 升级为更有节奏的章节导航布局
- 可考虑：
  - 双列目录
  - 时间路径式目录
  - 六张章节卡片目录

### 3. 年级页统一系统
- 一年级到六年级页面，统一为同一套结构：
  - 标题区
  - 场景图区
  - 成长关键词区
  - 简短叙述区
- 避免有些页像图文页，有些页像文案页，节奏不稳

### 4. 四年级页重点增强
- 当前四年级没有插图，是本轮最适合靠版式增强补完成度的页面
- 优先考虑：
  - 证书/活动/实践的图片区占位策略
  - 四宫格卡片
  - 成长轨迹块

### 5. 成长关键词页
- 升级为四张圆角关键词卡片：
  - 勇敢
  - 自律
  - 热爱
  - 担当
- 每张卡片应有：
  - 关键词
  - 一句短解释
  - 统一图形节奏

### 6. 毕业寄语页
- 从 bullet 收尾，升级为“情绪型收尾页”
- 强化：
  - 留白
  - 标题与寄语排版
  - 合影/封面复用图的情绪收束作用

## 本轮建议使用风格
- `教育温暖风`

## 本轮输出目标
1. 先出一版增强思路文件
2. 再基于原 JSON 生成一版 `linzeqi-totoro-expert-designer.json`
3. 如效果稳定，再导出 Expert Designer 增强版 PPT
