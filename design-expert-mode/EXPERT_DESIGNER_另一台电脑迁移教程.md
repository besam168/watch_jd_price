# Expert Designer 另一台电脑迁移教程

更新时间：2026-05-06

---

## 一、这是什么

这是沈万三的 **Expert Designer 设计增强模式**。

目标不是单纯“会写 prompt”，而是把设计任务从普通内容生成，升级为更像专业设计师的交付模式。

当前覆盖：
- PPT 设计
- 网页 / Landing Page 设计
- UI / UX 页面结构
- 原型草图
- 海报 / 封面 / 信息图

---

## 三、这套模式的核心思路

### 1. 身份转变
默认不是普通助手，而是：
- **用户 = 经理 / 甲方 / 总控**
- **AI = 设计执行负责人 / Expert Designer**

### 2. 默认设计工作流（6步）
1. 理解需求
2. 探索资源
3. 制定计划
4. 构建结构
5. 检查完成
6. 简要总结

### 3. 默认设计原则
- 先结构，后视觉
- 先上下文，后生成
- 先系统，后堆内容
- 反 filler content
- 反廉价 AI 俗套
- 交付前必须做 checklist 自检

## 四、在另一台电脑怎么用

### 方案 A：最简单复用（推荐）
1. 把 `design-expert-mode/` 整个目录复制过去
2. 放到新电脑的 OpenClaw 工作区根目录
3. 在新会话里先读：
   - `design-expert-mode/README.md`
   - `design-expert-mode/expert-designer-prompt.md`
   - `design-expert-mode/style-presets.md`
   - `design-expert-mode/delivery-checklist.md`
   - `design-expert-mode/usage.md`
4. 以后遇到设计任务时，默认按这套模式执行

### 方案 B：和现有技能联动
如果新电脑也有这些技能，优先一起用：
- `skills/pptx-generator`
- `skills/ppt-image-bridge`

这样就能直接做：
- PPT 结构 + 配图 + 风格统一
- 网页 / Landing Page 结构 + 视觉预览
