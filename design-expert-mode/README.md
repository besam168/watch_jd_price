# OpenClaw Expert Designer

把设计任务从“能做完”升级到“更像专业设计师交付”的本地设计增强模式。

## 目标

让沈万三在以下任务中默认进入 **Expert Designer** 模式：
- PPT 设计
- 网页设计 / Landing Page
- UI / UX 页面结构
- 原型草图
- 海报 / 封面 / 信息图

## 组成

1. `expert-designer-prompt.md`
   - OpenClaw 版设计增强提示词
2. `style-presets.md`
   - 常用设计风格预设
3. `delivery-checklist.md`
   - 交付前检查清单
4. `usage.md`
   - 如何触发与使用

## 默认设计流程（6步）

1. 理解需求
2. 探索资源
3. 制定计划
4. 构建结构
5. 检查完成
6. 简要总结

## 当前边界

- 重点先提升设计任务的结构化与美感稳定性
- 默认优先产出 HTML/CSS 预览结构、PPT 页面结构、图文协同方案
- 不宣称已具备完整动画引擎或完整前端工程流水线
