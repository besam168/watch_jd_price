# Style Presets

## 1. 商务高级灰
- 关键词：克制、专业、汇报感、可信、董事会风格
- 配色：深灰 / 石墨黑 / 冷白 / 少量蓝
- 适用：商业汇报、投资说明、项目方案
- 版式：大标题 + 卡片 + 数据块 + 充足留白

## 2. 科技蓝白
- 关键词：科技、未来、清晰、产品感、轻硬核
- 配色：蓝 / 白 / 银灰 / 深海军蓝
- 适用：产品介绍、技术说明、AI 工具、平台主页
- 版式：模块化分区、图标块、流程线、发光点缀

## 3. 教育温暖风
- 关键词：亲和、成长、治愈、清新、手绘友好
- 配色：森林绿 / 米白 / 暖黄 / 浅棕
- 适用：成长册、校园主题、亲子内容、教育海报
- 版式：圆角卡片、图文并排、留白柔和、标题温暖

## 4. 极简发布会风
- 关键词：高级、留白、品牌、简洁、聚焦主视觉
- 配色：黑 / 白 / 单一点缀色
- 适用：发布会 PPT、品牌页、单品展示页
- 版式：大图 + 大标题 + 极少文字 + 强节奏感

## 5. 国风雅致
- 关键词：留白、书卷气、东方、平衡、雅致
- 配色：米白 / 墨黑 / 黛青 / 暗红
- 适用：文化项目、国风品牌、文旅海报、传统主题页面
- 版式：上下分区、章法感、细线、留白、印章式点缀
