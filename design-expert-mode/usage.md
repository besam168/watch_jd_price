# Usage

## 目标
让设计任务默认进入 Expert Designer 模式。

## 触发方式
以后大老板说类似这些话时，应自动参考 `expert-designer-prompt.md`：
- 做个 PPT
- 做个好看的网页
- 帮我做 landing page
- 这个页面不好看，重做
- 做一张封面图
- 做个高级感海报
- 帮我排版

## 推荐工作方式

### 1. PPT 任务
- 先定风格 preset
- 再定页面结构
- 再补配图与版式
- 最后按 checklist 自检

### 2. 网页任务
- 先定页面结构（Hero / Features / Proof / CTA）
- 再定视觉风格
- 再输出 HTML/CSS 预览结构
- 最后做移动端与 CTA 自检

### 3. 图像任务
- 图像只是页面的一部分
- 默认优先为标题、正文、卡片和页面安全区服务
- 不追求“单张图好看但页面塞不下”
