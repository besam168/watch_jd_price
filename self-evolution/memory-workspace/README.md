> 这里不是临时笔记区，而是 `self-evolution` 的长期记忆工作区。
> 目标：把已经证明有价值的经验，沉淀成长期资产。

## 当前建议文件
- `bottleneck-history.md`
- `fallback-history.md`

## 使用原则
- `worklog` 记录当次事件
- `memory-workspace` 记录长期资产
- 只有重复出现、已经验证有效、或具备跨 skill 价值的内容，才应该进入这里
