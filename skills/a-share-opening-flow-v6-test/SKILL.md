# A股开盘风向与实时盯盘插件 V6（测试版）

## 当前命名
- 正式中文名：**A股开盘风向与实时盯盘插件 V6（测试版）**
- 简称：**V6 测试版**
- 技能目录：`a-share-opening-flow-v6-test`

## 定位
V6 测试版不是替代正式版。

它是在 **V6 正式版早盘强势判断** 基础上，再加入以下过滤条件：
- 近3日成交量整体放大
- 近3日股价整体拉升
- 5日线只做宽松辅助过滤

一句话：

**V6测试版 = V6正式版早盘强势判断 + 近3日放量拉升过滤**

## 目标
筛出这种票：
- 今天早盘已经被市场确认强势
- 属于最强板块或强势分支
- 同时近3个交易日日线量价配合转强
- 更像“盘中强 + 日线强”的共振票

## 输出建议
- 共振核心
- 共振跟随
- 盘中强，但日线不足

## 建议脚本
```powershell
python {baseDir}/scripts/opening_flow_v6_test.py --help
```
