# 名人技能模板说明

这个模板用于快速复制出一套“名人风格 + 方法论”型 skill。

## 用法

1. 复制整个目录
2. 把目录名改成新 skill 名
3. 修改 `SKILL.md` 里的占位符
4. 按研究文件模板补资料
5. 先做纯文档版验收
6. 只有在需要外部数据、自动化流程、结构化输出时，再补脚本或 API

## 默认适合的对象

- 咨询型名人
- 点评型名人
- 商业分析型名人
- 写作/表达型名人
- 访谈/演讲型名人

## 不适合直接照搬的对象

- 严重依赖私密经历或不可公开信息的人
- 需要执照/资质才能给建议的专业人士（医疗、法律、投资）
- 高度依赖实时数据流和数据库计算的角色

## 最佳实践

- 先抓骨架：价值排序、判断框架、流程结构
- 再补皮肤：口吻、节奏、句式、压迫感/亲和感
- 资料宁可少而硬，不要多而杂
- 有争议的人物，要把争议与边界单独列出来
