# A股开盘风向与实时盯盘插件 V6（正式版）

用于 A 股竞价与开盘前 5 分钟风向识别。

## 当前命名
- 正式中文名：**A股开盘风向与实时盯盘插件 V6（正式版）**
- 简称：**V6 正式版**
- 技能目录：`a-share-opening-flow`

## 目标
在以下两个窗口，识别最强方向与龙头：
- 9:20-9:25：竞价最后五分钟
- 9:30-9:35：开盘前五分钟真实承接

## 输出目标
- 最强板块 TOP3
- 每个板块的龙头个股
- 板块是联动走强，还是单票异动
- 竞价强但开盘掉队的伪强信号

## 第一阶段原则
当前版本优先做“实战够用版”，不吹逐笔精确资金流。

默认做法：

```powershell
python {baseDir}/scripts/opening_flow.py --help
```

## 口径
- 如果拿不到标准资金流字段，就先用竞价强势股/开盘强势股 + 板块归类近似反推
- 明确区分“竞价强”与“开盘后仍强”
- 不编造板块资金流数据
