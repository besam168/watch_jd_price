# a-share-opening-flow

A股开盘风向与实时盯盘插件 V6 正式版。

## 作用
用于盘中实时判断：
- 今日热门板块
- 热门个股
- 大资金关注股票
- 共振核心 / 共振跟随
- 上午主看名单

这是当前主用版本。

## 当前状态
已从骨架版修复为可运行正式版，具备真实抓取能力与中文正常输出。

## 入口
```bash
python scripts/opening_flow.py --json
python scripts/opening_flow.py
```

## 输出内容
- `top_sectors`
- `first_round_candidates`
- `passed`
- `partial`
- `failed`
- `resonance_core`
- `resonance_follow`
- `data_sources`

## 当前稳定数据源策略
- 东方财富：默认禁用
- 新浪 / 腾讯：盘中实时主源
- 腾讯历史K线（akshare）：日线过滤
- 板块热度：新浪 / 腾讯候选池分组 fallback

## 使用场景
- 盘中手动看市场
- 找热门板块
- 找强庄流入个股
- 做上午主看名单

## 判断逻辑
### 第一轮
先抓热门股和热门板块。

### 第二轮
做三项过滤：
- 近3日放量
- 近3日价格抬升
- 5日线宽松辅助

### 输出重点
- 共振核心：三项全过
- 部分通过：通过2项
- 不通过：弱于当前筛选要求

## 常见故障
### 1. 候选池为空
优先检查：
- 新浪接口是否可用
- 交易时段是否正常
- 本机网络是否正常

### 2. 板块为空
当前默认不依赖东方财富，板块来自新浪 / 腾讯候选池分组 fallback。
若仍为空，优先看候选池是否为空。

### 3. 中文乱码
当前正式版已经修复 UTF-8 输出。
如果换电脑后再出现乱码，优先检查：
- Python 输出编码
- 终端编码
- `PYTHONIOENCODING`

## 换电脑使用提醒
- 先确认 Python 可用
- 先确认 akshare 可用
- 默认不要重新打开东方财富主链
- 优先保持当前“新浪 / 腾讯主跑”策略

## 当前建议
正式盘中优先使用本插件，不要优先用测试版。
